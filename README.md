# Global-Mobile-Sales-Analytics-Dashboard
This project presents an interactive Mobile Sales Analytics Dashboard built using Power BI. The dashboard analyzes global smartphone sales data for 2025 and provides actionable insights into revenue, profit, sales trends, and customer behavior.

Key Objectives
1.Analyze total revenue and profit performance
2.Track monthly sales and profit trends
3.Compare brand-wise units sold
4.Visualize country-level sales distribution
5.Understand customer ratings and payment preferences

Key Insights
1.Total Revenue: $5M+
2.Units Sold: 5K+
3.Total Profit: $1.86M
4.Average Customer Rating: 4.02
5.Strong sales presence across multiple countries
6.Clear monthly trends in both revenue and profit

Tools & Technologies
* Power BI
* Data Modeling
* DAX Measures
* KPI Cards
* Map Visualizations
* Interactive Filters
